# CSS コーディングルール

- スタイルは Tailwind CSS ユーティリティクラスを優先して使用する
- カラー・フォントなどのテーマ変数は `src/css/theme.css` の `@theme {}` に定義する
- グローバルな CSS は `src/css/main.css` に記述する
- 基本的に**標準クラス**もしくは**定義済み変数**（`theme.css` の `@theme`）を使用する
- 固有値（任意値・`[...]` 記法）を使うよりも、標準クラス・定義済みクラスで近似することを優先する
- 近似した際は、どのように近似したかをコメントで明記する
